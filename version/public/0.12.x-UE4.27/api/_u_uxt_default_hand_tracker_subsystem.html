    UUxtDefaultHandTrackerSubsystem

    Subsystem for registering the default hand tracker.

    This subsystem creates the default hand tracker on player login. It registers input action mappings and binds to input events for Select/Grip actions. It also updates MotionControllerData of the default hand tracker once per world tick.
